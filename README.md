## 🚀 Project Portfolio

### 📁 **Project-1: Handwritten Letter Recognition**
**Neural Networks & Computer Vision**
- **Technology**: Multi-Layer Perceptron (MLP) with scikit-learn
- **Dataset**: EMNIST (60,000 training + 10,000 testing images)
- **Achievement**: Built a neural network that recognizes handwritten letters from 28x28 grayscale images
- **Real-world Application**: Converts handwritten text to typed format
- **Key Skills**: Image preprocessing, neural network architecture, model evaluation

### 📁 **Project-2: Sentence Prediction with LSTM**
**Natural Language Processing & Deep Learning**
- **Technology**: PyTorch LSTM-based encoder-decoder
- **Dataset**: VlogBrothers scripts data
- **Achievement**: Language model that generates coherent sentences based on initial prompts
- **Features**: Temperature-based sampling, word frequency analysis, text generation
- **Key Skills**: NLP, LSTM networks, text preprocessing, language modeling

### 📁 **Project-3: TrashBlaster AI Game**
**Reinforcement Learning & Neuroevolution**
- **Technology**: PyGame + Genetic Algorithms + Neural Networks
- **Approach**: AI agent learns through neuroevolution (200 neural networks evolving)
- **Achievement**: John Green Bot learns to navigate environment and destroy trash objects
- **Features**: Parallel training, fitness-based evolution, real-time visualization
- **Key Skills**: Reinforcement learning, genetic algorithms, game AI, parallel processing

### 📁 **Project-4: Movie Recommendation System** ⭐
**Machine Learning & Collaborative Filtering**
- **Technology**: Scikit-learn, Surprise (SVD), TF-IDF, Cosine Similarity
- **Dataset**: TMDB 5000 Movie Dataset + MovieLens ratings (90,000+ movies)
- **Achievement**: Hybrid recommender combining content-based and collaborative filtering
- **Features**: 100 movie recommendations, genre-based filtering, user preference analysis
- **Key Skills**: Recommendation systems, feature engineering, model evaluation

### 📁 **Project-5: Fruit Image Classifier** ⭐
**Deep Learning & Computer Vision**
- **Technology**: TensorFlow/Keras CNN
- **Dataset**: Fruits-360 (90,000+ images, 131+ fruit varieties)
- **Achievement**: 95%+ training accuracy, 90%+ validation accuracy
- **Features**: Real-time predictions, nutritional information integration, standardized pipeline
- **Key Skills**: CNN architecture, image classification, data augmentation, model deployment
